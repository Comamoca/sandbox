## 🚀 使い方

```
cd tailwind

# npm
npm install

# yarn 
yarn install

# pnpm
pnpm install

# watchしながらビルドします
npx tailwindcss -i input.css -o ../style/output.css --watch
```

## ⬇️  Install

# degitを使用してcloneします
