## 🚀 How to use

### Requires

- [deno](https://deno.com/)
- [bun](https://bun.sh/)
- [go](https://go.dev/)
- [hyperfine](https://github.com/sharkdp/hyperfine)


Use `deno task`
```sh
# run all task.
deno task all 

# only run build.
deno run build

# show other tasks
deno task
```
