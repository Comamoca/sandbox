## 🚀 How to use

Before, you must install dlang.
Show more info is [here](https://tour.dlang.org/tour/ja/welcome/install-d-locally).

```sh
# build & run
dub # or dub run

# to single binary
dub build

# build as release
dub -b release
```

