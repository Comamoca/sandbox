# nanojsx-ssg

自作SSGの試作品。試作段階の時は随時更新されます。試作品なのでもちろんコードとかは汚いです。


## 動かし方

```sh
deno run -A ./render.tsx
```

実行すると`_dist`に生成されたHTMLが出力されます。

```
_dist/
├── blog
│   └── index.html
└── index.html
```
